# 基于SpringBoot的物流管理系统+uniapp微信小程序

#### Description
基于SpringBoot的物流管理系统包含三部分：

前台客户下单等操作(微信小程序)
后台物流基本信息维护等操作(PC exe管理系统or B/S管理系统)
快递员扫码枪收发件等操作(微信小程序)
附带项目文档说明和部署视频。
本篇文章来源于学途拾珍( 源码：https://gitee.com/learning-journey-treasures)
